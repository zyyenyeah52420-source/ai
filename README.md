# 给屈丹晴的告白网页

这是一个特别为屈丹晴制作的感人网页，用来表达道歉和表白。

## 文件说明

- **index.html** - 主网页文件，包含所有页面内容
- **style.css** - 样式文件，负责网页的外观和动画
- **script.js** - 脚本文件，处理页面导航和交互效果

## 网页功能

网页包含6个页面：

1. **开场页面** - 精美的欢迎界面，带有跳动的爱心
2. **道歉页面** - 真诚的道歉信息
3. **回忆页面** - 回顾美好的时光
4. **表白页面** - 最深的爱意表达
5. **承诺页面** - 对未来的承诺
6. **庆祝页面** - 庆祝接受的时刻（有烟火和纸屑效果）

## 如何使用

### 方法1：直接打开文件
1. 用浏览器直接打开 `index.html` 文件
2. 点击按钮浏览各个页面

### 方法2：使用本地服务器
如果直接打开出现问题，可以使用Python开启本地服务器：

```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000
```

然后在浏览器中访问：`http://localhost:8000`

### 方法3：使用VS Code的Live Server扩展
1. 在VS Code中安装 Live Server 扩展
2. 右键点击 index.html
3. 选择 "Open with Live Server"
